# devtoprak text editor v.1

- *This program is a very lightweight text editor that can run even on the most basic systems.*  

## LICENSE

**This project is licensed under the Apache 2.0 License.**

## How do you use it?  

‘’’bash  
Sudo apt install python3-tk  
**Then, run the program**  
‘’’bash  
python3 texteditor.py  

## Features

- **txt and markdown supported,**
- **Save As feature,**  
- **Matrix theme**
